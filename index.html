🎯 Objectif du Projet

  • 💻 Interface HTML, CSS, JavaScript
  • 🔄 Communication WebSocket (Node.js)
  • 📦 Déploiement avec Docker

🛠️ Fonctionnalités Implémentées

  • ✅ Envoi de messages texte
  • 🖼️ Envoi d’images
  • 📝 Texte + image ensemble
  • 🙋‍♂️ Pseudo personnalisé + photo de profil
  • ⏱️ Affichage date + heure des messages
  • 🌗 Thème clair / sombre dynamique
  • 🐳 Docker : build + run du service complet

🧩 Architecture Technique

PartieTechnologie

Back-endNode.js (serveur WebSocket)
