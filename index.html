SAE 2.03 – Déploiement d'un Service Réseau

Projet : Chat en Temps Réel avec Docker

Présenté par : Ton Nom

Date : Avril 2025

💡 Infos

  • Classe : BTS SIO SLAM
  • Durée : 6 semaines
  • Technos : HTML, CSS, JS, WebSocket, Docker
"Ce projet m'a permis de comprendre l'importance de l'automatisation des services réseau."

🎯 Objectif du Projet

Concevoir et déployer un service de chat en temps réel à l'aide de :

  • Node.js + WebSocket pour le serveur
  • Interface HTML/CSS/JavaScript
  • Déploiement dans un conteneur Docker

🛠️ Fonctionnalités Implémentées

  • Envoi de messages texte en temps réel
  • Envoi d'images
  • Texte + image dans un seul message
  • Pseudo, photo de profil (ou défaut)
  • Date/heure d'envoi
  • Thème clair / sombre
  • Déploiement avec Docker

🧩 Architecture du Projet

  • Front-end : HTML, CSS, JavaScript (WebSocket)
  • Back-end : Node.js
  • Déploiement : Docker (basé sur Debian)

Cette SAE m’a permis de :
